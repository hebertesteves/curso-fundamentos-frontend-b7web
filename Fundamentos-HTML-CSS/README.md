# 📘 Curso HTML e CSS — Fundamentos

Repositório criado para armazenar os códigos, exemplos e práticas desenvolvidas durante o curso **Fundamentos em HTML/CSS**, da **B7Web**.

Este projeto tem como objetivo consolidar os conceitos fundamentais de **HTML e CSS** no desenvolvimento web, com foco em estruturação de páginas, estilização e boas práticas de organização do código.

---

## 🧠 Conteúdos abordados

Durante o curso, foram estudados os seguintes tópicos:

- Conceitos básicos de HTML e estrutura de páginas  
- Estrutura básica de documentos HTML  
- Uso de **tags e atributos**  
- Introdução ao **CSS e estilização de páginas**  
- Formatação de textos com HTML e CSS  
- Cores no CSS  
- Bordas no CSS  
- Margins e Padding no CSS  
- Width e Height em elementos HTML  
- Criação de **links**  
- Listas no HTML  
- Tabelas no HTML e CSS  
- Criação de **formulários HTML**  
- Tipos de **inputs do HTML5**  
- Select, Textarea, Button e Label  
- Estilização de formulários com CSS  
- Inclusão de **fontes externas**  
- Comentários em HTML  
- Conceitos de **herança, especificidade e cascata no CSS**  
- Uso de **unidades de medida absolutas e relativas**
